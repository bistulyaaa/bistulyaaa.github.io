快速上手针孔摄影

发表于2025-09-25|更新于2025-09-25
|浏览量:

本文章用于知道如何快速上手一种古老的摄影技术——针孔摄影

没错就是那个小学科学课学的

针孔摄影最核心的物理学原理是：光在均匀介质中沿直线传播。

想象一个黑暗的房间里，只在墙壁上开了一个小孔。当室外的景物被阳光照亮时，景物上任意一点（例如树顶的A点）都会向四面八方反射光线。在这些光线中，只有一束能恰好穿过小孔，照射到房间内对面的墙壁上，形成A’点。

同理，景物底部的B点反射的光线，也只有一束能穿过小孔，照射到内壁的B’点。其他所有点的光线也以此类推。

结果是：房间内对面墙壁上会形成一个倒立的、左右相反的室外景物的影像。这个现象被称为 “小孔成像”。

针孔相机的基本构成

一个最简单的针孔相机只需要三个部分：

一个密闭的暗箱： 阻止任何不需要的光线进入。

一张感光材料： 安装在暗箱的内壁上，用于记录图像（传统相机是胶片，现代DIY的可以是相纸，数码的则是图像传感器）。

一个极小的孔： 这就是“针孔”。它取代了传统相机复杂的镜头，是成像的关键。

针孔成像的关键特性

无限景深

    这是针孔摄影最显著的特点。由于没有镜头对焦，从几厘米远的物体到无限远的景物，在照片中都能保持清晰。你不需要“对焦”，所有距离都是同步清晰的。

图像是倒立的

    正如原理部分所述，因为光是直线传播的。来自物体上部的光线穿过小孔后，只能投射到感光材料的下部；反之，来自物体下部的光线则投射到上部。因此成像必然是倒立的。

针孔的大小与图像质量的关系

    孔太大：如果孔开得太大，从一个物点发出的多条光线都能通过孔，在感光材料上会形成一个模糊的光斑（而不是一个清晰的点）。多个光斑重叠，会导致图像非常模糊、不清晰。这就像是每个像素都被“抹开”了。

    孔太小：当孔非常小时，成像会变得更清晰。但小到一定程度时，会发生光的衍射现象。光波会绕过孔的边缘散开，导致光线相互干扰，图像反而会再次变得模糊。

    最佳孔径：因此，存在一个“最佳孔径”，它需要在“光线直射成像”和“衍射效应”之间取得平衡。这个最佳直径可以通过公式估算，通常与暗箱的尺寸（焦距）有关。一个常用的经验公式是：最佳孔径 ≈ √(焦距 / 25)，其中焦距和孔径单位均为毫米。例如，焦距100mm的暗箱，最佳孔径约为√(100/25) = √4 = 2mm。

更精准的三角函数方法：
焦距=（针孔直径/0.03679）2
使用直角三角法求解视角。简化的公式为：
视角 = tan -1 ( (d/2)/(t/2) ) x 2
图像直径 (mm) = 2 x 焦距 x tan ((视角)/2)

**曝光时间极长**

    针孔非常小，进光量小。因此，针孔摄影通常需要很长的曝光时间，从几秒到几个小时不等。这使得它非常适合拍摄如流动的云、平静的水面等景物，而动的人物则会变成模糊的鬼影甚至消失。

怎么把这个像开玩笑的东西变成正经器材

答案是：减少衍射为了减少衍射，应尽可能使针孔边缘整洁。不一定非常圆，方孔和长方形孔同样很有效。针孔边缘的金属越薄，衍射就越少。因为激光钻出的针孔会留下了一个隧道，除非金属足够薄，如果你希望得到一个更理想的针孔，就要用600号砂纸打磨针孔区域对于摄影来说，微钻针孔与激光针孔相比同样锐利。只有当针孔附近的材料足够薄时激光针孔才会更出色。
我建议你可以买0.04mm的铜带，没错 0.04。接着，购买一支手捻钻，大概十几块。与其搭配一组钨钢钻头，从0.01mm到0.9mm
然后，减少反射。铜片以及相机内部要用好一点的消光棉做消光。
